## Recommended directories (free to submit)

| Directory | Submit URL | Notes |
|-----------|------------|--------|
| **AITrendyGame** | https://aitrendygame.com/submit-a-game | Free, 1–2 day approval. Expect Chrome/Firefox/Safari/Edge, mobile-friendly, fast load. |
| **itch.io** | https://itch.io/games/new | Create a project, choose “HTML” and add your game URL or embed. Great for indie visibility. |
| **HTML5 Gamers** | https://html5gamers.com/submit-game | HTML5-only. Submit only finished games; search first to avoid duplicates. |
| **Indie Games Developer** | https://devcentral.indiegamesdeveloper.com/submit-your-game-free-form | Free form for indie devs; you can add trailer, press kit, release status. |
| **CrazyGames** | https://developer.crazygames.com/ | Developer portal for HTML5/Unity; can earn revenue. Check their requirements before submitting. |

## Other options (check requirements first)

- **Poki** (https://sdk.poki.com) – Very large audience but often requires **web exclusivity** (game only on Poki on the open web). Only consider if you’re willing to make the game exclusive there.
- **Kongregate** – No longer accepts new game submissions.

---

## Quick checklist before submitting

- [ ] Game works in Chrome, Firefox, Safari, Edge
- [ ] Playable on mobile (responsive or mobile-friendly)
- [ ] Page loads in under ~10 seconds
- [ ] No broken links on the main play page
- [ ] `og-image.png` (or your chosen image) loads at: https://mahjongboss.com/og-image.png

## After submitting

- Some directories allow you to add analytics or track referrals; check each site’s developer/partner docs if you care about traffic sources.
